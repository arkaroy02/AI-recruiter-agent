📊 SCORING SYSTEM
═══════════════════════════════════════════════════════════════════════════════

Match Score (60% weight)
    ├─ Required Skills: 55%
    ├─ Preferred Skills: 25%
    └─ Experience Fit: 20%

Interest Score (40% weight)
    ├─ Genuine enthusiasm
    ├─ Availability
    └─ Red flags

Combined = (Match × 0.6) + (Interest × 0.4)

EXAMPLE:
┌─────────────────────────────────────────┐
│ Alice Chen: 87.4/100 ⭐ TOP CHOICE     │
│                                         │
│ Senior Full Stack Engineer              │
│ Match: 85/100 | Interest: 92/100       │
│ "1-3 months" availability               │
│ Strong signals of genuine interest      │
└─────────────────────────────────────────┘

📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: Connection refused
   → Start Ollama: ollama serve

Issue: Model not found
   → Download: ollama pull gemma2:2b

Issue: Out of memory
   → Reduce top_k or add RAM

Issue: Port 8501 in use
   → Use different port: streamlit run app.py --server.port 8502

Issue: Python errors
   → Run: python test_demo.py (full diagnostics)

More help?
   → See QUICKSTART.md troubleshooting section
